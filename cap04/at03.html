Seção 2 — Ranking de Tecnologias

  1. HTML

    Linguagem responsável pela estruturação de páginas web.

  2. CSS

    Utilizado para estilizar páginas e melhorar a experiência visual.

  3. JavaScript

    Permite criar interatividade e dinamismo em aplicações web.

  4. Node.js

    Plataforma utilizada para executar JavaScript no servidor.

  5. React

    Biblioteca popular para construção de interfaces modernas.

Seção 3 — Glossário de Termos Técnicos

HTML
Linguagem de marcação utilizada para estruturar páginas web.
CSS
Linguagem utilizada para estilização visual de páginas web.
JavaScript
Linguagem de programação utilizada para interatividade.
DOM
Estrutura em árvore que representa um documento HTML.
API
Interface de comunicação entre sistemas e aplicações.
Pode ser utilizada para integração entre serviços web.
HTTP
Protocolo utilizado para transferência de dados na internet.
Base da comunicação entre navegador e servidor.
URL
Endereço Web
Localização de um recurso disponível na internet.
DNS
Sistema de Nomes de Domínio
Serviço responsável por converter domínios em endereços IP.
